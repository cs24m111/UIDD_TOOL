# Assets Folder

This folder should contain the following image files for the Instagram clone to work properly:

## Required Images:

1. **avatar.jpg** - Profile picture/avatar (recommended: 200x200px)
   - Used for user avatars in posts and stories
   - Can be any portrait image

2. **post1.jpg** - First post image (recommended: 600x600px or similar aspect ratio)
   - Example: Nature/landscape photo
   - Used by user "nature_lover"

3. **post2.jpg** - Second post image (recommended: 600x600px or similar aspect ratio)
   - Example: Travel/destination photo
   - Used by user "travel_enthusiast"

4. **post3.jpg** - Third post image (recommended: 600x600px or similar aspect ratio)
   - Example: Food photo
   - Used by user "foodie_delight"

## Image Specifications:

- Format: JPG (JPEG)
- Recommended dimensions: 600x600px for posts, 200x200px for avatars
- File size: Keep under 1MB for faster loading
- Quality: Medium to high quality

## How to Add Images:

1. Find or create 4 images matching the descriptions above
2. Rename them exactly as listed (avatar.jpg, post1.jpg, post2.jpg, post3.jpg)
3. Place all files in this `assets/` folder
4. Refresh the browser to see the images in the Instagram clone

## Placeholder/Testing Options:

If you don't have images ready, you can:
- Use free stock photos from sites like Unsplash, Pexels, or Pixabay
- Use placeholder image services like:
  - https://via.placeholder.com/600x600.jpg
  - https://picsum.photos/600/600
- Create simple colored squares using image editing software
- Use screenshots or any personal photos

**Note:** Ensure you have the rights to use any images you add to this project.
