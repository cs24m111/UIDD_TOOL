What This Tool Checks

This tool analyzes social media platforms for compliance with 6 key IT Rules 2021 amendments:

  • Rule 2(1)(wa): Definition of "synthetically generated information"
  • Rule 4(2): Automated tools for detecting harmful synthetic content
  • Rule 4(4): Complaint handling for AI-generated content
  • Rule 3(1)(b): Prohibition of harmful deepfakes and misleading AI content
  • Rule 3(3): Labeling requirements (10% coverage, metadata, immediate identification)
  • Rule 4(1A): SSMI requirements (50 lakh+ users)
